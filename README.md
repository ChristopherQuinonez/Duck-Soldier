# Duck Soldier

Created by Christopher Quinonez and Nicholas Aghedo

Duck Soldier is a 2D wave survival game created in Unity.
## Gameplay
The player controls a duck defending against waves of human poachers.  
Melee enemies attack the barrier while ranged enemies shoot from lanes.

## Controls
- Left Arrow → Move Left
- Right Arrow → Move Right
- Spacebar → Shoot
- R → Restart after Game Over

## Features
- Wave-based enemy spawning
- Melee and ranged enemy types
- Barrier defense system
- Main menu and settings
- Credits and how-to-play screen
- Sound effects and music
